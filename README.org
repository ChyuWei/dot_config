#+TITLE: dot_config

* 输入法

[[file:rime/][Rime 万象拼音]] — macOS Squirrel 鼠须管配置。

安装：

#+begin_src bash
cd rime && python3 rime.py install
#+end_src

日常改配后部署：

#+begin_src bash
cd rime && python3 rime.py config
#+end_src

| 文件 | 说明 |
|------+------|
| [[file:rime/squirrel.custom.yaml][squirrel.custom.yaml]] | 鼠须管外观（主题、字体、字号、圆角） |
| [[file:rime/default.custom.yaml][default.custom.yaml]] | 候选数 & 翻页键 |
| [[file:rime/rime.py][rime.py]] | 一键安装 / 更新 / 软链部署脚本 |

* 终端 & 工具

| 配置 | 说明 |
|------+------|
| [[file:_config/ghostty/config][ghostty]] | 终端模拟器 |
| [[file:_config/tmux/tmux.conf][tmux]] | 终端复用器 |
